## Open
```
Super | Super+A         Shows the application menu
Ctrl+Alt+T              Terminal
Alt+F2                  Run a command
Super+M                 Toggle notification tray
```

## Navigate
```
Ctrl+Alt+arrow          Move between workspaces
Super+Tab | Alt+Tab     Switch between running applications
Alt+Esc                 Quick switch between apps
Super                   Opens Activities search
Super+D | Ctrl+Alt+D    Show desktop
Super+Arrow keys        Snap windows
```

## Session
```
Super+L | Ctrl+Alt+L    Locks the screen
Ctrl+Alt+Del            Log out
```
